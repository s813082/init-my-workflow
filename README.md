## 🚀 如何使用 (How to use)

換了新電腦後，打開終端機並執行以下指令：

```bash
# 1. 下載你的工作流 (記得換成你自己的 Git Repo 網址喔！)
git clone https://github.com/你的帳號/init-my-workflow.git ~/Documents/init-my-workflow

# 2. 進入資料夾並啟動安裝
cd ~/Documents/init-my-workflow
chmod +x install.sh
./install.sh
```

---

## ⚠️ 最後的手動步驟 (Manual Steps)

安裝完後，還有兩個小動作要請笨蛋弟弟親自動手：

1.  **設定字體**: 
    - 開啟 iTerm2 > `Settings` > `Profiles` > `Text`。
    - 在 `Font` 選擇 **MesloLGS NF**。
2.  **設定配色**: 
    - 開啟 iTerm2 > `Settings` > `Profiles` > `Colors`。
    - 在 `Color Presets` 選擇 **Tomorrow Night Eighties**。
3.  **調整主題**:
    - 在終端機輸入 `p10k configure` 重新調整你的漂亮箭頭！
