init-my-workflow
核心安裝 軟體百科

打造最強 macOS 開發環境

換新機時不再手忙腳亂。這是一套專為開發者打造的自動化工具包，透過智慧偵測與自癒同步，一鍵建立您的夢幻工作流。

立即部署 探索軟體百科

$ git clone https://github.com/s813082/init-my-workflow.git

$ cd init-my-workflow && ./install.sh

🔍 正在為您全面偵測系統環境...

✅ 已就緒：Google Chrome, iTerm2, LINE...

📦 待徵召：VSCode, Stats, Keka...

_

一鍵啟動指令

git clone https://github.com/s813082/init-my-workflow.git ~/Documents/init-my-workflow && sudo chown -R $(whoami) /usr/local/share/man/man8 && cd ~/Documents/init-my-workflow && chmod +x install.sh && ./install.sh
指令已複製！✨
