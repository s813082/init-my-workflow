Obsidian + GitHub Automation

打造最強 第二大腦 連動機制

這不僅是一個筆記軟體的安裝，而是一套深度的知識流自動化方案。我們將 Obsidian 的本地優勢與 GitHub 的雲端同步完美結合。

智慧同步

透過 GitHub CLI (gh) 實現無縫認證，無論新舊機都能一鍵拉取完整筆記庫。

資料防禦

內建路徑偵測機制，防止覆蓋現有筆記，確保您的寶貴知識資產絕對安全。

開箱即用

自動修改 Obsidian 系統設定檔，完成後直接開啟即可看到所有 Vault。

核心技術亮點

Technique 01

Python 結構化 JSON 處理

我們捨棄了脆弱的字串替換，改用內嵌 Python 指令來解析與重建 `obsidian.json`。這能精準處理 JSON 樹狀結構，支援「追加 (Append)」與「覆蓋 (Overwrite)」兩種模式，確保設定檔永遠不遺失。

Technique 02

GitHub CLI (gh) 深度整合

利用 `gh repo clone` 取代傳統的 git 指令，腳本能自動處理私有倉庫的權限與認證，大幅降低使用者的操作門檻。

Technique 03

冪等性設計 (Idempotency)

腳本執行前會檢查 Homebrew 目錄、Git 遠端關聯以及檔案系統路徑。不論重複執行幾次，腳本都能正確識別目前狀態，避免產生冗餘配置。

如何使用？

1

執行部署腳本

在終端機輸入 `./setup-obsidian.sh` 啟動互動式嚮導。

2

認證與 Clone

選擇選項 1 並輸入您的倉庫名稱，其餘的自動化邏輯會幫您搞定。

3

開啟即享受

直接打開 Obsidian，您的第二大腦已經在那裡等您了。
